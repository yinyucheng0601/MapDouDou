# 拼豆图纸生成器

一个纯前端本地 Web App，用来把图片自动转换成拼豆图纸。

## 在线体验

[GitHub Pages 预览](https://yinyucheng0601.github.io/MapDouDou/)

## 功能

- 上传图片后自动生成拼豆图纸
- 启动时默认加载根目录的 `sample.png`
- 宽度范围支持 `24-96` 颗豆子
- 提供常用预设宽度按钮
- 自动识别边缘背景并裁切主体
- 两种颜色策略：
  - 内置拼豆色板近似
  - 保留图片主色
- 自动统计每种颜色需要多少颗
- 自动估算 `29 x 29` 底板数量
- 导出图纸 PNG、圆珠预览 PNG、符号矩阵 CSV

## 运行

这个项目是静态文件，但因为使用了 ES module，建议用本地 HTTP 服务打开：

```bash
cd /Users/yin/bead-pattern-app
python3 -m http.server 4321
```

然后访问：

`http://127.0.0.1:4321`

## 适用场景

- 像素角色
- 小图标
- Figma 导出的纯色插画
- 需要快速做成拼豆参考图纸的图案

如果图里有复杂渐变、阴影或照片纹理，建议先在 Figma 里做一版简化配色，再导入这个应用。
